# 3주차 과제 (B811186)

3주차 정리

* git branch란 : 분기 (포인터)

보고 있는 공간을 가르키는 header

처음 branch 이름 : master

가장 마지막 커밋을 가르키는 포인터 : testing

$git checkout testing으로 header를 testing에 옮긴다

$git status 내가 어디에 있는지확인하는 명령어

$git branch testing : testing 이라는 이름의 브랜치를 만든다

$git branch : branch들 확인
(*붙은곳이 내가 보고있는 branch)

* merge와 conflict

merge
